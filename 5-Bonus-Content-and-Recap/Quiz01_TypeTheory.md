# Quiz 01 - Type Theory

Exercises

1. Lambda calculus recap
   - Reduce a simple lambda expression using beta reduction.
   - Explain the difference between a variable, an application, and an abstraction.

2. Types and inference
   - State the principal type of a few small Haskell expressions.
   - Explain what type inference gives you in a language like Haskell.

3. Typed lambda calculus
   - Give the type of a simple lambda term.
   - Explain why types rule out some nonsensical programs.

4. Hindley-Milner
   - Describe, at a high level, how Hindley-Milner supports type inference.
   - Explain why polymorphism is useful.
