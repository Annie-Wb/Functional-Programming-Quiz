# Quiz 03 — Concurrency

Exercises

1. Threads and `forkIO`
   - Write a program `twoThreads` that spawns two threads which print alternating messages.

2. `MVar` and synchronization
   - Implement a simple producer/consumer using `MVar`.

3. `STM` and transactions
   - Implement a concurrent counter using `TVar` and `STM`.
