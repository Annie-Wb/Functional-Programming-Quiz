Functors, Applicatives, Monads, and Monadic Parsing
=====================================================

This folder contains quizzes and examples covering Functors, Applicatives, Monads, and monadic parsing.

Files:
- `Quiz01_Functors.md`
- `Quiz02_Applicatives.md`
- `Quiz03_Monads.md`
- `Quiz04_MonadicParsing.md`
- `Solutions.md`
- `src/` — skeleton/example modules: `FAMExamples.hs`, `ParserExamples.hs`.
- `test/` — law checks and parser demos: `FunctorLawsTest.hs`, `ParserExample.hs`.

Load modules in `ghci` to experiment:

```bash
ghci -isrc test/ParserExample.hs
```
